How to use

フォームを作成する

maltonn.github.io/form/input?{各種パラメータ}

Parameter list

パラメーター 説明 デフォルト値
sid サービス名 ※必須
t タイトル -
q{int} {int}番目の質問のid ※必須
q{int}lg 質問のタイトル -
q{int}ph 質問のPlaceholder -
q{int}tp 質問の入力タイプ text

Example

https://maltonn.github.io/form/input?sid=example&t=form1&q1=name&q1lg=What's your name

Example2

https://maltonn.github.io/form/input?sid=example2&t=form2&q1=name&q1lg=your name&q2=age&q2lg=your age&q2t=number

集計結果を見る

maltonn.github.io/form/table?{各種パラメータ}

Parameter list

パラメーター 説明
sid サービス名
{id} 回答のフィルターに使う

Example1

https://maltonn.github.io/form/table?sid=example

Example2

https://maltonn.github.io/form/table?sid=example2&age=16
